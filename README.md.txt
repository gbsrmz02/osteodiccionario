# 🦴 OsteoDiccionario v1.0
Diccionario osteológico completo para Criminología y Antropología Forense, con imágenes y datos forenses.

## 🛠️ Tecnologías usadas
- HTML5, CSS3, JavaScript
- Node.js + Express
- Python + FastAPI
- MySQL + PostgreSQL
- Docker + Docker Compose
- CMD, PowerShell, Linux

## 🚀 Ejecución
1. Tener instalado **Docker Desktop**
2. Abrir terminal en carpeta raíz
3. Ejecutar:
   - Windows: `scripts\iniciar_windows.cmd`
   - PowerShell: `.\scripts\iniciar_powershell.ps1`
   - Linux: `bash scripts/iniciar_linux.sh`

## 🌐 Acceso
- Interfaz web: http://localhost:8080
- API Node: http://localhost:3000/api/huesos
- API Python: http://localhost:5000/docs
